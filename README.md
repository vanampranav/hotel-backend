# Hotel Management Backend

A Spring Boot backend application for managing hotel information and bookings.

## Features

- User authentication with JWT
- Hotel management (add, update, delete)
- User management with role-based access control
- Email notifications

## Technologies

- Spring Boot 3.2.3
- Spring Security
- Spring Data JPA
- MySQL Database
- JWT Authentication
- Gradle

## Deployment

This application is configured for deployment on Railway.app
